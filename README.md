# Sobre o projeto: Leitor de textos 

o leitor de textos é uma aplicação em Javascript puro par auxiliar crianças no desenvolvimento da comunicação.

## Tecnologias usadas:

* HTML 
* CSS
* Javascript
* Web Speech API
